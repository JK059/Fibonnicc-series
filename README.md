The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones, usually starting 
from 0 and 1 (0, 1, 1, 2, 3, 5, 8, 13...). It is defined by the formula \(F(n)=F(n-1)+F(n-2)\). 
This project implements this sequence to demonstrate recursive, iterative, or dynamic programming approaches,
